## Convention adopter :
Pas de '\_' -> les '\_' représentent les espaces dans la suite

- Variable : mot1Mot2Mot3
- Noeud : Mot1\_Mot2\_Mot3
- Type : Mot1Mot2Mot3
- Script : Mot1Mot2Mot3
